KIEN THUC VE GIT

1.	Git và Github là khác nhau, Git là một phần mềm, Github là một trang web. Phần mềm Git có thể up code lên nhiều web khác nhau, trong đó có github là một trong số đó.
2.	Dữ liệu từ folder cho vào stack rồi mới cho lên kho lưu trữ git
3.	Kho lưu trữ: github.com/tienxuantt/cv.github.io.git
4.	CÁC BƯỚC LÀM VIỆC VỚI GIT
5.	Mở project rồi chuột phải rồi chọn git bash here
6.	Tên miền: cv.github.io
7.	Git init : khởi tạo kho stack trong máy để lưu trữ
8.	Git add “index.html”  : để cho file index vào stack  
9.	Git add . : add tất cả các file 
10.	git remote add origin https://github.com/tienxuantt/cv.github.io.git
11.	 git status : kiểm tra trạng thái trong stack
12.	Git commit –m “upload 2 file”:  kí tên 
13.	git push -u origin master  : cho lên kho lưu trữ trên mạng
14.	git log : xem cac giai đoạn để quay ngược thời gian
15.	ctrl C : dừng xem log
16.	git checkout **** : 4 mã đầu tiền
17.	git checkout master : quay ve lan cuoi commit
